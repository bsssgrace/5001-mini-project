1.ผลกระทบของโควิด Impact ราคาโภคภัณฑ์ในช่วงแรกทำให้ราคาตกลงในช่วงต้น(ราคาข้าวเปลือก) จนถึงปลายq1ของ2020
2.หลังจากปลายq1ของ2020 เริ่มไม่เห็นความสัมพันธ์สินค้าโภคภัณฑ์กับจำนวนผู้ติดเชื้อ
